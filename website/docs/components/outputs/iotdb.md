## 使用说明

IoTDB 输出组件允许您将数据写入 Apache IoTDB 时间序列数据库。该组件支持以下功能：

1. 配置 IoTDB 服务器地址、用户名和密码
2. 设置连接超时时间
3. 批量写入数据以提高性能
4. 配置最大并发连接数

请注意，此组件需要能够访问正在运行的 IoTDB 服务器实例。
